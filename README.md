# BensonLabs

Applied systems, infrastructure, and practical engineering work.

This organization is used to manage personal infrastructure, homelab environments, and operational tooling. Repositories are maintained with an emphasis on reproducibility, clarity, and real-world use.

## Focus Areas

- Containerized infrastructure (Docker, Compose)
- Endpoint management workflows (Intune, JAMF)
- Local AI systems and experimentation
- Small-scale service architecture

## Philosophy

Build systems that can be rebuilt.

Avoid throwaway work. Favor clarity over cleverness. Everything here should be usable, repeatable, and grounded in real workflows.
